# Ноутбуки проекта

Актуальные notebooks для итогового проекта:

```text
01_eda.ipynb
02_modeling.ipynb
```

`01_eda.ipynb` выполняет EDA по реальному Jira Dataset: загрузку данных, проверку структуры, пропусков, типов признаков, базовую статистику, формирование целевой переменной и построение графиков.

`02_modeling.ipynb` загружает реальные результаты экспериментов из `artifacts/leaderboard.json`, формирует таблицу сравнения моделей и строит график метрик.

Файл `01_eda_and_experiments.ipynb` оставлен только как совместимость со старой структурой и перенаправляет к актуальным notebook-ам. Финальный проект использует реальный Jira Dataset, а не синтетические данные.
