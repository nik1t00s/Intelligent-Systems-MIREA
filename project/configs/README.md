# Конфигурация проекта

Основной конфигурационный файл итогового проекта:

```text
configs/config.yaml
```

В нем описаны:

- имя проекта и `random_seed`;
- путь к исходному датасету;
- путь к подготовленной таблице;
- целевая переменная;
- список числовых, категориальных и бинарных признаков;
- train/test split;
- основная метрика;
- параметры моделей `LogisticRegression`, `RandomForest` и `GradientBoosting`.

Файл `train_config.yaml` оставлен как совместимая копия старого имени конфига. Актуальный код по умолчанию читает `configs/config.yaml`, если переменная окружения `CONFIG_PATH` не переопределена.
